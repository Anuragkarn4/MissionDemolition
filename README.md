# MissionDemolition

Unity Version: 2021.3.33f1
Course: CS382 – Game Design, Development, and Technology
Author: Anurag Karn

This project is an enhanced version of the Mission Demolition game.
The player uses a slingshot to launch projectiles and destroy castles across multiple levels.

The game progressively increases in difficulty and includes visual, audio, and gameplay improvements.

# How to Play
- Move mouse over the slingshot → Launch point appears
- Click and drag → Aim and control power
- Release mouse → Launch projectile
- Destroy all castles to advance levels
- Complete all levels → Game Over screen appears
- Click Play Again to restart

# Enhancements
1. Game Over System
2. Multiple Levels (Progressive Difficulty)
3. Rubber Band (Line Renderer)
4. Sound Effect
5. Extra Enhancement (Trajectory Line)

# Technologies Used
1. Unity Engine (2021.3.33f1)
2. C#
3. Unity UI System (Canvas, Buttons)
4. Line Renderer
5. Rigidbody Physics
6. AudioSource
